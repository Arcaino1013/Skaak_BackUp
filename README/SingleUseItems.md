## Allgemein:
- Versteinern:
    - Du kannst eine Figur außer dem König auf dem Spielfeld für einen Zug immobilisieren, sodass sie sich nicht bewegen kann. Sie kann im darauffolgenden Zug nicht geschlagen werden.
- Glasskanone:
    - Wenn du einen Turm oder einen Läufer auswählst, und dieser eine Figur schlägt, wird auch die Figur dahinter je nach Bewegungsrichtung direkt geschlagen für diesen Zug. D
      abei verlierst du jedoch auch deine Figur. Sollte der König hinter einer Figur stehen, ist dies Schachmatt.
- Schutzschild:
    - Du kannst eine Figur entsprechend ihres Tiers auswählen. Diese Figur hat in dieser Runde 2 Leben. 
      Wird sie angegriffen, wird die angreifende Figur in ihre Ausgangsposition zurückgesetzt. Es gibt vier Tiers: 
        - Tier 1 - Bauern
        - Tier 2 - Springer und Läufer
        - Tier 3 - Turm
        - Tier 4 - König oder Dame
- Märtyrer: 
    - Wenn eine Figur mit dem Effekt 'Märtyrer' geschlagen wird, wird auf ihrem Feld eine Bombe platziert. Steht beim nächsten Zug des Gegners eine Figur darauf, wird sie ebenfalls geschlagen
